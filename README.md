# NGINX-Agent

## Required
### Ubuntu/Debian
(`sudo apt-get install`)
- build-essential
- libssl-dev
- libffi-dev
- python-dev
- python-pip

### Fedora/RHEL-derivatives
(`sudo yum install`)
- gcc
- libffi-devel
- python-devel
- openssl-devel
- python-pip

## Python packages required
To install all at once:
```
pip install -r requirements.txt
```
